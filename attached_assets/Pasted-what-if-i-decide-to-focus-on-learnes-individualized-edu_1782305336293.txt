Part 1 — Learner Assessment & Performance Level Placement
This is the heart of your system. Getting placement right determines everything else.

Assessment Framework — Multi-Dimensional
Do not assess on academics alone. Assess across 6 dimensions:

Dimension	What You Measure	Why It Matters
Academic ability	Literacy, numeracy, subject knowledge	Core learning level
Learning style	Visual, auditory, kinesthetic, reading	How they absorb best
Cognitive ability	Memory, attention, problem solving, processing speed	Capacity to learn
Socio-emotional level	Self-regulation, motivation, confidence	Affects performance
Special needs screening	Dyslexia, ADHD, autism indicators	Requires different approach
Language proficiency	Mother tongue vs instruction language	Critical in Kenya
Assessment Methods (Built Into the System)
1. Adaptive Digital Assessment
Learner takes an interactive online test
Questions adjust automatically based on answers
Gets harder if answered correctly, easier if wrong
System identifies exact level within 20–30 questions
Works on tablet, phone, or computer
Can be done offline and synced later
2. Observational Assessment (Teacher Input)
Teacher fills structured observation checklist in system
Rates learner on behavior, participation, social skills
Flags concerns like attention issues or emotional challenges
System combines this with digital test results
3. Portfolio Assessment
Upload samples of learner's work over time
Drawings, written work, project photos
AI analyzes progression and flags gaps
Especially important for CBC competency tracking
4. Parent Input Assessment
Parent answers structured questions about child at home
Sleep patterns, behavior, home language, health issues
Socioeconomic indicators that affect learning
Flags if child needs additional support beyond academics
5. Screening Tools (Built-In)
Basic dyslexia screening
Attention and focus indicators
Fine motor skill assessment (for young learners)
Emotional wellness check-in (age appropriate)
Performance Level Structure
After assessment, learner is placed into one of 5 performance bands:

Level	Name	Description
Level 1	Foundational	Needs intensive basic support — literacy/numeracy not yet developed
Level 2	Developing	Below grade level, making progress with support
Level 3	Approaching	Near grade level, needs targeted intervention
Level 4	Proficient	At grade level, learning independently
Level 5	Advanced	Above grade level, needs enrichment & extension
Each level has:

Specific learning objectives
Assigned resources (soft & hard copy)
Recommended teaching strategies
Suggested parent support activities
Clear milestones to move to next level
Re-Assessment & Movement Between Levels
Trigger	What Happens
Every school term	Automatic re-assessment scheduled
Teacher flags progress	Ad hoc assessment triggered
Parent requests review	Assessment initiated
Learner completes all milestones	System prompts level-up assessment
Sudden performance drop	System alerts teacher and parent
